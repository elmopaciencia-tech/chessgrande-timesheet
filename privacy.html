Chess Grande Payroll Portal Privacy & Security Notice

This portal is for authorized Chess Grande employees and managers only.

Authentication

  • Sign-in is handled by Supabase Auth.
  • This app does not store raw passwords in local storage or custom password tables.

Payroll Data

  • Payroll submissions are stored in Supabase with Row Level Security policies.
  • Only managers can access all submissions; employees can access their own records.

Banking Details

  • Only enter production bank details in trusted production environments.
  • For testing environments, use non-sensitive test data only.
